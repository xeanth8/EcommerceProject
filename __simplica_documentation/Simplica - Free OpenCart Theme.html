Installation

Installation for OpenCart 2.2.0.0
  • 1Make sure you have installed and running the correct version of OpenCart
  • 2It is a good practice to backup your system
  • 3Open the folder that corresponds to your OpenCart version and upload all the files from "FILES_TO_UPLOAD" to the root directory you have installed OpenCart in.

  • 4 Theme Activation
    • Go to Extensions > Themes > Edit and choose simplica from the Theme Directory selector. See example.
  • 5 Go to Extensions > Modifications and click on the blue Refresh button (top right corner) for update the system. See example.
  • 6Congratulations! Installation is complete. Go back to your main store front to view the template installed on your site.
Installation for OpenCart 2.3.0.2 / 3.x
  • 1Make sure you have installed and running the correct version of OpenCart
  • 2It is a good practice to backup your system
  • 3Open the folder that corresponds to your OpenCart version and upload all the files from "FILES_TO_UPLOAD" to the root directory you have installed OpenCart in.

  • 4 Theme Activation
    • Go to Extensions > Extensions > Selecting "Themes"(from Choose the extension type)> then Edit Your store > and choose simplica from the Theme Directory selector. and then Save. See example.
  • 5 Go to Extensions > Modifications and click on the blue Refresh button (top right corner) for update the system. See example.
  • 6Congratulations! Installation is complete. Go back to your main store front to view the template installed on your site.

Adjust your image sizes

Before you start using your store with the simplica theme, image settings must be edited with the following values:

  • Login to your Opencart Admin Area
  • Go to Extensions > Themes > Edit.
  • For Opencart 2.3.0.2 / 3.x - Go to Extensions > Extensions > Selecting "Themes"(from Choose the extension type) > Edit default store theme.

Here, We have set image size in vertical dimesions. Images don't have to be vertical, you can use square dimesions and proportions. for example 260x260.

Default Slider

  • 1First, create group of banners
    • Go to Design > Banner
    • Click on Insert button. Pick a name for your banner SlideShow for example, make sure it is enabled and click on "Add Banner".
    • Pick a title and link for the first slider image then choose the image itself (Browse Files). Make sure your images are the appropriate size: 1140x380 Repeat this step for as many slider Items you want to have then click SAVE
  • 2Go to Extensions > Modules > Slideshow > install & Edit
  • 3Go to Extensions > Modules > Slideshow > Edit and from the banner dropdown select the previously created "SlideShow" banner. Put in the following values for slider size shown on the image below to make sure your slider fills up the appropriate space: 1140x380. For Example:

 

Featured, Latest, Bestsellers, Specials Modules

featured products, latest products, bestsellers, Specials modules, etc. adjust the image settings with the following recommended values:

For Square image Dimension set the dimensions to 260x260
